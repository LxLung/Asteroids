# Pygame Asteroids
Shoot rocks and stuff!
# Installation Instruction
1. Get the Repo

Download Zip or clone it

3. Setup Venv

python3 -m venv venv

On linux: source venv/bin/activate

or

On Windows: venv/Scripts/activate.bat

4. Install pygame dependency

pip install -r requirements.txt

5. Run the game!

make

or

python ./main.py
